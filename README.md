# isweb++
### Fast & Lightweight HTTP/s checker

### Usage
```
Usage: isweb++ [host file]

```
### Main Features
     - Pass a list of subdomains or IPs and get the HTTP/s served ones along with their status code
     - Multithreaded
     - Easy to use
